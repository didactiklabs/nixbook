# AI Agents Guide

This repository contains configuration for NixOS.

## Tools
- `nixpkgs`
- `colmena`
- `npins`

## Structure
- `nixosModules/`: NixOS modules
- `homeManagerModules/`: Home Manager modules
- `customPkgs/`: Custom packages
- `npins/`: npins sources
- `profiles/`: Machine profiles
- `assets/`: Static assets and plugins
- `hive.nix`: Colmena configuration
- `base.nix`: Base configuration for all machines
